O jogador possui os seguintes comandos:
◈ w: O jogador se movimenta para cima
◈ a: O jogador se movimenta para esquerda
◈ s: O jogador se movimenta para baixo
◈ d: O jogador se movimenta para direita
◈ i: Interação Jogador/Objeto

O jogo possui 8 tipos de elementos:
◈ 웃: Jogador
◈ ✱: Parede
◈ @̵ :Chave na qual o jogador, ao interagir(i), irá abrir uma porta
◈ ⟦⟧: Porta fechada
◈ ⟦ ⟧: Porta aberta
◈ O: Botão no qual o jogador deve passar por cima para abrir uma porta
◈ ⧼#⧽: Espinho
◈ >: Teleporte onde o jogador irá viajar de um lugar a outro e vice-versa

O jogador só possui 5 vidas, após a perda de todas as vidas, é game over
